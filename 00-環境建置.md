# Codex 懶人包 #00：環境建置

> 版本：v0.1（Codex 版）
> 更新日期：2026-04-26

> 📌 **本懶人包可獨立執行**：會自動檢查並安裝所需工具。

---

## 這個懶人包會幫你做什麼？

把使用 Codex 懶人包所需的所有環境一次建置完成：
- 安裝 **OpenAI Codex CLI**
- 安裝 Git、GitHub CLI、uv
- 登入 OpenAI、登入 GitHub
- 驗證所有工具都能正常使用

---

## 先備條件

- [ ] 有 OpenAI 帳號（ChatGPT Plus / Pro / Business 或 API 額度）
- [ ] 有 GitHub 帳號（免費即可）
- [ ] 電腦有網路連線
- [ ] 已安裝 Node.js 18 以上（沒裝沒關係，步驟一會裝）

---

## 請 Codex 幫我執行以下步驟

> ⚠️ 把這份 MD 檔貼給 Codex CLI，它會自動執行。
> 遇到需要手動操作的地方會暫停指示你。

---

### 步驟零：環境檢查

請 Codex 執行以下檢查，並把缺少的項目記錄下來：

1. **作業系統**：Windows / macOS / Linux
2. **網路連線**：ping 一個外部網站
3. **Node.js**：`node --version`（需要 18+）
4. **Git**：`git --version`
5. **GitHub CLI**：`gh --version`
6. **uv**：`uv --version`

---

### 步驟一：安裝 Node.js（如果未安裝或版本 < 18）

**Windows**：
```bash
winget install --id OpenJS.NodeJS --accept-source-agreements --accept-package-agreements
```

**macOS**：
```bash
brew install node
```

**Linux（Ubuntu/Debian）**：
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

確認：`node --version`

---

### 步驟二：安裝 OpenAI Codex CLI

```bash
npm install -g @openai/codex
```

確認：
```bash
codex --version
```

> 找不到 `codex` 指令請重開終端機。

---

### 步驟三：登入 Codex

```bash
codex
```

第一次執行會出現登入畫面，**建議選「Sign in with ChatGPT」**（用 ChatGPT 帳號登入，會走 OAuth），免去管理 API Key。

> 🖐️ **手動操作**：瀏覽器會開啟 OpenAI 授權頁面，登入並授權後回到終端機。

也可以用 API Key：
```bash
export OPENAI_API_KEY="sk-..."
```

確認登入：對 Codex 說「hi」看是否回應。

---

### 步驟四：安裝 Git（如果未安裝）

**Windows**：
```bash
winget install --id Git.Git --accept-source-agreements --accept-package-agreements
```

**macOS**：`xcode-select --install`

**Linux**：`sudo apt update && sudo apt install git -y`

確認：`git --version`

---

### 步驟五：安裝 GitHub CLI（如果未安裝）

**Windows**：
```bash
winget install --id GitHub.cli --accept-source-agreements --accept-package-agreements
```

**macOS**：`brew install gh`

**Linux**：見 [GitHub CLI 官方安裝指引](https://github.com/cli/cli/blob/trunk/docs/install_linux.md)

確認：`gh --version`

---

### 步驟六：安裝 uv（如果未安裝）

uv 是 Python 套件管理工具，後續安裝 MCP 工具會用到。

**Windows（PowerShell）**：
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**macOS / Linux**：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

確認：`uv --version`

---

### 步驟七：登入 GitHub

```bash
gh auth login --web --git-protocol https
```

> 🖐️ **手動操作**：終端機會顯示驗證碼，瀏覽器開啟 GitHub 授權頁面，貼驗證碼後 Authorize。

確認：`gh auth status`

---

### 步驟八：設定 Git 使用者資訊

```bash
git config --global user.name "你的姓名"
git config --global user.email "你的email"
```

---

### 步驟九：最終驗證

```bash
codex --version
git --version
gh --version
gh auth status
uv --version
```

> 全部通過後告知使用者：「✅ 環境建置完成！你的電腦已經準備好使用所有 Codex 懶人包了。」

## 常見問題

| 問題 | 解法 |
|------|------|
| `codex: command not found` | npm 全域安裝路徑沒進 PATH，重開終端機；Windows 確認 `%APPDATA%\npm` 在 PATH |
| Codex 一直要 API Key | `Sign in with ChatGPT` 選項在第一次互動式啟動時出現；或設 `OPENAI_API_KEY` 環境變數 |
| winget 找不到 | Windows 10 1809+ 內建；或到 Microsoft Store 安裝「App Installer」 |
| Node.js 版本太舊 | 移除舊版後重新安裝最新 LTS |
